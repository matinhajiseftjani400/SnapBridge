# اسنپ‌بریج 🚀

**SnapBridge** یک وب‌اپلیکیشن سبک، سریع و بدون نیاز به نصب است که امکان ارسال و دریافت آنی تصاویر بین دو دستگاه را از طریق مرورگر و با استفاده از یک کد امن ۶ رقمی فراهم می‌کند.

---

## 📌 توضیح پروژه (GitHub Description)

ارسال و دریافت سریع تصاویر بین دستگاه‌ها از طریق مرورگر با استفاده از کد امن ۶ رقمی، بدون ثبت‌نام و بدون نصب.

## ✨ ویژگی‌ها

- 📸 ارسال تصویر از موبایل و دسکتاپ
- 🔢 اتصال امن با کد ۶ رقمی
- 📥 دریافت آنی تصاویر
- 🧹 پاک‌سازی خودکار فایل‌ها
- ⚡ سریع و بدون نیاز به دیتابیس
- 📱 کاملاً ریسپانسیو (موبایل‌فرندلی)
- 🔊 افکت صوتی هنگام ارسال و دریافت

---

## ⚙️ نصب و راه‌اندازی

1. پروژه را روی سرور یا لوکال‌هاست کپی کنید:
   ```bash
   /var/www/html/snapbridge
   ```

2. اطمینان حاصل کنید PHP فعال است (ترجیحاً PHP 7.4 یا بالاتر)

3. سطح دسترسی پوشه `temp` را تنظیم کنید:
   ```bash
   chmod -R 777 temp
   ```

4. پروژه را در مرورگر اجرا کنید:
   ```
   http://localhost/snapbridge
   ```
